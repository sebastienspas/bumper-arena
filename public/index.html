— multijoueur —
Multijoueur : Pour jouer ensemble, tous les joueurs doivent ouvrir ce lien et entrer le même code salle. Un serveur WebSocket (Colyseus / Socket.io) est nécessaire pour synchroniser les positions en temps réel. Ce prototype simule le multijoueur avec des bots IA.

Pour déployer le vrai multijoueur :
1. Serveur Node.js + Colyseus pour la synchro temps réel
2. Hébergement du serveur (Railway, Fly.io, ou VPS)
3. ~200 lignes de code serveur à ajouter
